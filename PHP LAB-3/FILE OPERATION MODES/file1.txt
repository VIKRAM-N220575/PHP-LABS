Task 3: File Operation Modes 
�
�
Problem Statement 
Demonstrate all file open modes using fopen() and explain their behavior. 
�
�
Students must write programs for: 
● r → Read only 
● w → Write only (erase old data) 
● a → Append only 
● x → Create new file (fail if exists) 
● r+ → Read & Write 
● w+ → Read & Write (erase old data) 
● a+ → Read & Append 
● x+ → Create new file for Read & Write
